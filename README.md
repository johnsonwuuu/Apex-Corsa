# Apex Corsa
https://youtu.be/RMFo3-H1-jI
Apex Corsa is a standalone Windows build of a Unity racing project.

## How to Run

1. Download or clone this repository.
2. Keep all files and folders together in the same directory.
3. Run:

```text
My project (3).exe
```

The executable depends on the included Unity runtime files, especially:

- `My project (3)_Data/`
- `MonoBleedingEdge/`
- `D3D12/`
- `UnityPlayer.dll`

Do not move the `.exe` away from these files, or the game may fail to launch.

## Contents

This repository contains the compiled Windows deliverable, including the Unity player, managed assemblies, runtime data, and supporting DLLs required to run the build.

## Platform

- Windows 64-bit
- Built with Unity

## Notes

This is a final build folder, not the full Unity editor source project. To edit the game in Unity, use the original Unity project files rather than this standalone build.
